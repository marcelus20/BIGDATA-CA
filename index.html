A brief explanation about bitcoin

Bitcoin is an innovative payment network and a new kind of money. Bitcoin uses peer-to-peer technology to operate with no central authority or banks; managing transactions and the issuing of bitcoins is carried out collectively by the network. Bitcoin is open-source; its design is public, nobody owns or controls Bitcoin and everyone can take part. Through many of its unique properties, Bitcoin allows exciting uses that could not be covered by any previous payment system.

Info about a specific Block

A block is a collenction of transactions performed by the network. Each block stores around 2500 transactions and after it has reached its store limit, it needs to be validated. Validation means that the new hash for this block needs to be discovered and this task is performed by the miners. When the block is validated, this new hash is linked with the next block "previous hash" attribute and here is where the blockchain comes from. The number of zeros in the begining is determined by the difficulty reate. The more defficult the more zeros there will be. In order to discover the hash of the block, there is not a better approach other than brute force combinations of nonces. It works like this:

  • Let the SHA256(currentBlockState) = x.
  • Network will apply the difficulty rate, so x will become 00000000x;
  • The idea is to look like random code, but the actual fact, it is not. It will give you the same output for a given input SHA256(input) = 00000000x;
  • So there is not a better aproach then guessing the value of nouce combined with the state of the block to get the 00000000000x result.
